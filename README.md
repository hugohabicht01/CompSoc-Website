# CompSoc Website

The website for the Lancaster University Computer Science Society.


## Project Setup

### System Dependencies

- hugo
- docker
- docker-compose

### testing

To test the static content run `hugo serve`.

To test the go http server run `go run .` from inside the `src` directory.

### docker build

`docker-compose up`

`docker-compose build`
